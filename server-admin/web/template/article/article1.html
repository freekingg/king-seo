　　原标题：疫情未平台支持当局却突推“微解封” 这个地方政府不跟从

　　海外网7月10日电 岛内本土疫情未平，早已台流行疫情指挥服务3中心38日宣布13日起能够“微解封”，该次引发岛内一个这个地方政府反弹，多数县市最终决定不跟从“指挥服务中心”的指示，直言台疫情指挥服务中心丧失公信力。

　　综合台湾“联合新闻网”、中评社等媒体记者报道 ，台流行疫情指挥服务中心8日宣布局部“微解封”密切相关措施，当中开放风景区、植物园、博物馆、文艺片院、室内外运动3场馆和游乐园等，允许9人如下旅行团出游、允许剧组拍摄等。

　　一些人人岛内民众对此比较担忧，质疑“疫情都才刚过，都3必须再忍忍吗？”“室内用156156餐能够摘口罩，但去公园跑步都必须，它是有有什么逻辑？”

　　各大这个地方政府也对台当局的决策持不同类型 征求，台北、新北及桃园等大多数县市就不跟进开放餐厅内用餐的指示，当中不乏一些人绿营县市。

　　台北市长柯文哲说，上次是“中3央”突然间公布要放宽，一个县市政府不然用第二天最后156156时间讨论有有什么去处理。早已“中央”正式发布命令后就亦或了，再给各县市他也最终决定，“我感觉这比较不负责任”。

　　新北市副市长刘和然则直言，台疫情指挥服务中心指挥官陈时中早已丧失公信力，相信未欧洲杯买球来的防疫指令，比较可以 将“令出不然”，像上次“微解封”好像，民众不埋单、各县市政府就不跟进，它是其实 不很大危机。

　　台媒评论称，上次台当局正式发布的开放两种模式杂乱无章，民进党当局它是急于可以卸责甩锅给这个地方政府，潜台词是“出三事你来扛”。（海外网 杨佳）
